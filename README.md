# Destroid

The one-bazillionth game inspired by the arcade classic, Asteroids. Destroid is written using C++17, OpenGL and an entity component system (ECS) architecture.

## Controls

<kbd>W</kbd> - Thrust

<kbd>A</kbd> <kbd>D</kbd> - Rotate

<kbd>Space</kbd> - Fire

<kbd>ESC</kbd> - Exit game
